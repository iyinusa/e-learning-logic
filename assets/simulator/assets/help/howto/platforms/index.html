Platform

There are a number of different ways to use or install the brainbox software. "brainbox" is an browser base application which can be served from a node.js server.

Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. The Node.js applications are written in JavaScript, and can be run within the Node.js engine. You need the Node.js environment if you want have access to any hardware GPIO or connect any wireless device...welcome to awesome the world of IoT.

Raspberry Pi

How to setup Node.js on a Raspberry Pi and run brainbox on it

1.) Setting up your Raspberry Pi

Running a basic setup of Raspbian on a Raspberry Pi is very easy and clearly explained in their Getting Started Guide

Download NOOBS, extract it on your SD, plug it in and turn on the Raspberry Pi. When prompted, choose to install Raspbian and let it run.
